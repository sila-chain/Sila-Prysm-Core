# Validator Deposit Contract Local Copy

This package contains a copy of the official Sila [Validator Deposit Contract](https://github.com/sila/consensus-specs/tree/e4a9c5fa29def20c4264cd860868f131d6f40e72/solidity_sila_deposit) along with its ABI, bytecode, and Go bindings generated by go-sila's [abigen](https://github.com/sila/go-sila/tree/master/cmd/abigen) `version 1.10.4-stable`. It contains useful test harnesses for setting up and deploying a validator sila deposit using Go bindings, which are used across tests in Sila.
